- ΔH°f [ZnCl₂(aq)] = -415.1 kJ/mol
- ΔH°f [HCl(aq)] = -167.2 kJ/mol
- S° [Zn(s)] = 41.6 J/(mol·K)
- S° [H₂(g)] = 130.7 J/(mol·K)
